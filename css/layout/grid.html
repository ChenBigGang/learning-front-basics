grid 布局

grid 容器可用属性

  • grid-template-rows 定义行，包括每行的高度，并且可以给每一条线设置名称(1个或者多个),单位可以使用fr表示一份
  • grid-template-columns 定义列，包括每列的宽度，并且可以给每一条线设置名称(1个或者多个)
  • row-gap 行间距
  • column-gap 列间距
  • gap 行间距和列间距的简写
  • grid-template-areas 划定区域，给每个单元格起名字，可以说是划定集合，名称相同的单元格是同一个集合，可以一起引用
  • grid-auto-flow 指定元素放定的顺序，可用的值有 row|column|row dense|column dense 加上dense表示会尽量占满grid 容器
  • justify-items 容器中所有项目在水平方向上的对齐方式
  • align-items 容器中所有项目在垂直方向上的对齐方式
  • place-items 是上面两个属性的简写
  • grid-auto-columns
  • grid-auto-rows 这两个属性用来设置 浏览器自动创建的多余网格的列宽和行高
  • grid-template 是grid-template-columns、grid-template-rows和grid-template-areas的简写
  • grid 是 grid-template-rows grid-template-columns grid-template-areas grid-auto-rows grid-auto-columns grid-auto-flow的简写

gird 项目属性

  • grid-column-start
  • grid-column-end
  • grid-row-start
  • grid-row-end 就是指定项目的位置，注意指定的是边框，不是单元格。可以是数字 第几 也可以是名称
    也可以使用span表示跨域几个单元格 span number 如果项目出现重叠，可以使用z-index来控制层级关系
  • grid-column grid-column-start和grid-column-end的简写 比如：grid-column:1 / 3
  • grid-row grid-row-start和grid-row-end的简写 比如：grid-row: 1 / 4
  • grid-area 指定项目存放在哪个区域，grid-template-area 定义的区域，也可以用来简写 column和row 的start和end 四个属性 比如 grid-area: 1/1/3/3
  • justify-self 单元格内部的元素水平放置方法
  • align-self 单元格内部的元素垂直放置方法
  • place-self 是上面两个属性的简写
