display的block把元素设置为盒子模型，inline把元素设置为行内元素，盒子独占一行，可以设置宽高，行内元素不可以设置宽高 和其他元素独占一行，宽高有子元素内容撑开。inline-block可以让元素既可以设置宽高，又不独占一行，和float有很多相似之处

inline-block，子元素设置inline-block，会让元素在父容器中平铺

可以发现子元素之间有间隙，这是因为标签后面的回车换行符导致的，设置父容器fontsize=0可以消除空格。

float 会让元素也在父容器中平铺，但实际上此时子元素已经脱离了文档流，可以发现父元素高度为0

可以使用overflow:hidden 解决父元素的高度坍塌问题
